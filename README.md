##### Building and Emulating the Phonegap Application #####

Ensure Bower, Grunt, and Phonegap packages are installed globally. Suitable only for Mac environments with XCode.

1. cd into the "mobile" directory.
2. npm install
3. grunt init
4. grunt emulate
